无意间突发奇想，能否在终端中打印地图，然后找到了一篇文章介绍，只需运行这行代码：

```bash
$ telnet mapscii.me
# 若未安装telnet需安装一下
$ yum install telnet -y
$ sudo apt-get install telnet
```
